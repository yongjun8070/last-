# 심해 탐험 (Deep Sea Explorer) 🌊

심해의 신비를 탐험하는 3D 스타일 브라우저 게임입니다. 잠수정을 조종하여 깊은 바다로 내려가며 다양한 생물들을 발견하고 고대 유적을 탐사하세요!

## 게임 특징 ✨

- **3D 원근 효과**: Canvas를 활용한 유사 3D 그래픽
- **15종 이상의 다양한 심해 생물**: 깊이에 따라 다른 생물들이 출현
- **5개의 깊이 존**: 표층부터 초심해층까지
- **고대 유적 발견**: 깊은 곳에 숨겨진 신비한 유적들
- **압도적인 거대 생물**: 리바이어던, 고대의 존재 등
- **실시간 자원 관리**: 산소와 조명 관리
- **부드러운 물리 시뮬레이션**: 잠수정의 현실적인 움직임

## 조작법 🎮

- **← →** (또는 A, D): 좌우 이동
- **↑** (또는 W): 상승
- **↓** (또는 S): 하강
- **SPACE**: 조명 켜기/끄기
- **ESC**: 수면으로 빠르게 귀환

## 깊이 존 🌊

### 1. 표층 - Epipelagic Zone (0-200m)
- 밝고 평화로운 구역
- 물고기, 해파리, 바다거북

### 2. 중층 - Mesopelagic Zone (200-1,000m)
- 빛이 약해지는 황혼대
- 초롱아귀, 오징어, 손도끼물고기

### 3. 심층 - Bathypelagic Zone (1,000-4,000m)
- 완전한 어둠
- 아귀, 바이퍼피쉬, 주머니장어

### 4. 심해층 - Abyssopelagic Zone (4,000-6,000m)
- 극한의 압력
- 대왕심해등각류, 뱀파이어오징어, 거미물고기

### 5. 초심해층 - Hadal Zone (6,000m+)
- 알려지지 않은 세계
- 심연의 뱀, 리바이어던, 고대의 존재
- **고대 유적 출현**

## 생물 도감 📖

게임을 플레이하면서 다양한 생물들을 발견해보세요!

- 🐠 평화로운 물고기들
- 🪼 투명한 해파리
- 🐢 느긋한 바다거북
- 🐙 거대한 오징어
- 🔦 발광하는 심해어류
- 🦷 날카로운 이빨의 포식자들
- 🐍 신비로운 심해 뱀
- 👁️ 압도적인 거대 생물
- 🗿 고대의 유적

## 생존 가이드 💡

1. **산소 관리**: 깊이 잠수할수록 산소가 빨리 소모됩니다. 수면(깊이 50m 이내)으로 자주 올라와 보충하세요.

2. **조명 관리**: 조명을 켜면 주변을 잘 볼 수 있지만 배터리가 소모됩니다. 필요할 때만 사용하세요.

3. **생물 관찰**: 가까이 다가가면 생물을 발견할 수 있습니다. 하지만 큰 생물들은 위험할 수 있습니다!

4. **압력 주의**: 깊이 내려갈수록 압력이 증가합니다. HUD에서 현재 압력을 확인하세요.

5. **안전한 귀환**: 위험하다고 느껴지면 ESC 키로 빠르게 수면으로 돌아가세요.

## 실행 방법 🚀

### 로컬에서 실행

1. 저장소 클론 또는 파일 다운로드
2. `index.html` 파일을 브라우저에서 열기
3. "탐험 시작" 버튼 클릭

### 서버에서 실행

```bash
# Python으로 간단한 서버 실행
cd deep-sea-explorer
python -m http.server 8000

# 또는 Node.js http-server 사용
npx http-server
```

브라우저에서 `http://localhost:8000` 접속

## 기술 스택 💻

## 팁과 트릭 🎯

- 깊이 200m마다 새로운 생물들이 나타납니다
- 거대한 생물들은 플레이어를 따라옵니다
- 조명을 끄고 있으면 일부 생물들이 발광하는 것을 볼 수 있습니다
- 유적은 6000m 이하의 깊은 곳에만 나타납니다
- 최대한 많은 생물을 발견해보세요!
